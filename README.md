# 🎯 ConversIQ — SaaS Onboarding Flow Optimizer

> Built for **Zoho Cliqtrix 2025 National Hackathon** | Solo Build | Bot-Building Competition

---

## 🔴 The Problem
SaaS users sign up, get confused during onboarding, and silently leave —
before the support team even knows they're struggling. This is called **trial churn**.

## ✅ The Solution
ConversIQ detects when a trial user is **stuck** and automatically triggers
contextual help — before they give up.

---

## ⚙️ How It Works

| Step | Tool | What Happens |
|------|------|--------------|
| 1 | Zoho SalesIQ | Detects user inactivity on onboarding step (90-sec threshold) |
| 2 | Zoho Flow | Receives stuck-signal via webhook, triggers next action |
| 3 | Zobot | Opens contextual chat: *"Having trouble with Step 3? Here's how."* |
| 4 | Human Handoff | If unresolved in 2 exchanges → agent gets full context |

---

## 🛠️ Tech Stack
`Zoho SalesIQ` · `Zoho Flow` · `Zobot` · `JavaScript` · `Webhooks` · `HTML/CSS`
